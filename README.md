# docx2md

Convert Microsoft Word Document to Markdown

![docx2md](https://raw.githubusercontent.com/mattn/docx2md/master/screenshot.png)

## Usage

```
$ docx2md NewDocument.docx
```

## Installation

```
$ go install github.com/mattn/docx2md@latest
```

## Supported Styles

* Header
* HyperLink
* Indent
* Table
* List
* Bold
* Italic
* Strike
* Embeded Image
